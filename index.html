Descricao dos Indicadores

DPA - OCUPACAO

% de ocupacao produtiva do colaborador. Fonte: DPA Verint. Medicao: Bom para Cima.

CHAT TOA

% de interacao dentro de 10 min. Mede a agilidade de resposta no chat do sistema TOA.

ETIT POR EVENTO RAL

% tarefas direcionadas para o tecnico em ate 30min. Demanda tipo RAL (Reclamacao).

ETIT POR EVENTO REC

% tarefas direcionadas para o tecnico em ate 30min. Demanda tipo REC (Reconexao).

ETIT OUTAGE SEM SINAL (GPON)

Volume dentro do prazo (DP) / Volume Total. Mede eficiencia no tratamento de outages GPON.

LOG OUTAGE REPROGRAMADO SEM SINAL GPON

% outages reprogramados sintoma sem sinal GPON. Bom para Baixo.

ETIT OUTAGE RESIDENCIAL SEM SINAL

Data/Hora do primeiro acionamento que foi para o TOA. Mede tempo de resposta.

ETIT OUTAGE RESIDENCIAL DEGRADACAO

Data/Hora do primeiro acionamento que foi para o TOA. Mede tempo de resposta para degradacao.
